---
title: "List Airtime Networks"
api: "GET /airtimeList"
description: "Get all available airtime networks and their details"
---

## Overview

Retrieve a list of all available airtime networks with their pricing, limits, and other relevant information.

## Request

<RequestExample>
```bash
curl --location 'https://api.bleeprs.com/api/airtimeList' \
--header 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

## Parameters

No parameters required for this endpoint.

## Response

<ResponseField name="gatewayStatus" type="string">
  Status code indicating success ("00") or error
</ResponseField>

<ResponseField name="gatewayMessage" type="string">
  Human-readable status message
</ResponseField>

<ResponseField name="requestReference" type="string">
  Unique reference ID for tracking this request
</ResponseField>

<ResponseField name="data" type="array">
  Array of available airtime networks
</ResponseField>

### Data Object Fields

<ResponseField name="ID" type="number">
  Unique identifier for the network
</ResponseField>

<ResponseField name="Network" type="string">
  Network name (e.g., "MTN", "AIRTEL", "GLOBACOM", "9MOBILE")
</ResponseField>

<ResponseField name="Operator" type="string">
  Operator type (e.g., "AIRTIME")
</ResponseField>

<ResponseField name="Region" type="string">
  Geographic region (e.g., "MTN", "AIRTEL")
</ResponseField>

<ResponseField name="MinAmount" type="number">
  Minimum airtime amount that can be purchased
</ResponseField>

<ResponseField name="MaxAmount" type="number">
  Maximum airtime amount that can be purchased
</ResponseField>

<ResponseField name="Percentage" type="number">
  Commission percentage for this network
</ResponseField>

<ResponseField name="NetworkLogo" type="string">
  Network logo identifier
</ResponseField>

<ResponseField name="VoucherCode" type="string">
  Voucher code for this network (e.g., "LA_MTN")
</ResponseField>

<ResponseField name="CreatedAt" type="string">
  Timestamp when the network was added
</ResponseField>

<ResponseField name="UpdatedAt" type="string">
  Timestamp when the network was last updated
</ResponseField>
