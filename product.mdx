### Go-live Checklist

The following checklist should be considered when going live on Bleeprs

| Go-live Activities     | Description                                |
| ---------------------- | ------------------------------------------ |
| `Request Sender ID`    | Request a sender id                        |
| `Fund Account Balance` | Topup account using stripe                 |
| `Request Coverage`     | Request access to a given coverage, region |
